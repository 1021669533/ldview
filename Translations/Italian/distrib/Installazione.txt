Per poter usare questo file di lingua bisogna copiare i file "LDView-Italian.dll" e "Guida.html" nella cartella, nella quale è stata installato LDView.

Se la lingua del vostro computer non è impostata su "Italiano", potete usare il file "LDView-Italian.reg" per forzare LDView a caricare "LDView-Italian.dll come file di lingua. Basta cliccare due volte su "LDView-Italian.reg" e rispondere "Sì" quando si viene chiesto se si vuole aggiungere le informazioni del file "LDView-Italian.reg" al registro del sistema. In caso si volesse riportare LDView alla lingua predefinita (Inglese) basta cliccare due volte sul file "LDView-DefaultLanguage.reg".
