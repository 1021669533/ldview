Smooth Curves Examples

The following shots were all taken to illustrate the curve smoothing option. Please note that this option only smooths the shading of curved surfaces. It doesn't add new polygons. This can be seen by looking at the edges, which remain the same whether the option is enabled or not. (This is particularly evident with the motorcycle helmet.)
