Screenshots

I have taken a number of shots of the same model from (nearly) the same position with different settings to give you an idea of some of the different ways you can view a model. All of the shots were taken using the high quality stud files. I have included PNG versions of all the full-sized screenshots.


This shot was taken with the settings set to values which produce a traditional 3D image. Lighting is turned on, and edge highlights are turned off. The seam width is set to 0.50, and primitive substitution is enabled.

This is the same shot, but with edge highlights turned on, and seam width set to 0.
